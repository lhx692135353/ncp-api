**接口文档**

**根据身份证查询信息接口**

接口地址：/find-by-id-card
请求方式：GET

请求字段：

| 字段名称 | 字段类型 | 是否必填 | 标识   |
| -------- | -------- | -------- | ------ |
| IDCard   | string   | 是       | 身份证 |


返回数据：

| 字段名称  | 字段类型 | 标识                            |
| --------- | -------- | ------------------------------- |
| errorCode | int      | 返回200为成功,其他为失败        |
| message   | string   | 请求返回消息                    |
| status    | int      | 状态 1、首次录入 2、 第二次录入 |
| name      | string   | 姓名                            |
````
{
  "errorCode": 200,
  "message": "",
  "attachment":{
    "status":1,
    "name":"刘帅"
  }
}
````
