**采集人员登录接口**

接口地址：/login

请求方式：POST

请求字段：

| 字段名称 | 字段类型 | 是否必填 | 标识   |
| -------- | -------- | -------- | ------ |
| phone    | String   | 是       | 电话号 |
| password | String   | 是       | 密码   |
|          |          |          |        |

返回数据：

| 字段名称  | 字段类型 | 标识                     |
| --------- | -------- | ------------------------ |
| errorCode | int      | 返回200为成功,其他为失败 |
| message   | string   | 请求返回消息             |
| cpId      | int      | 录入人员唯一ID           |
| cpName    | String   | 录入人姓名               |
| csName    | String   | 所属采集点               |
| csId      | int      | 所属采集点id             |

````
{
    "errorCode": 200,
    "message": "登陆成功",
    "attachment": {
        "cpId":1,
        "cpName":"陈老师",
        "csName":"路边",
        "csId":1
    }
}
````
