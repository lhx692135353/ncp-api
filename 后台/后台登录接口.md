**后台登录接口**

接口地址：/manage/login

请求方式：POST

请求字段：

| 字段名称 | 字段类型 | 是否必填 | 标识                 |
| -------- | -------- | -------- | -------------------- |
| userName   | String  | 是       | 电话号        |
| userPassword | String  | 是       | 密码 |
|          |          |          |                      |

返回数据：

|字段名称|字段类型|  标识   |
|----|-----|------|
|errorCode|int|返回200为成功,其他为失败|
|message|string|请求返回消息|
````
{
    "errorCode": 200,
    "message": "登陆成功",
}
````
