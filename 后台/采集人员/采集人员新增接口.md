**采集点新增接口**

接口地址：/manage/insert-cp

请求方式：POST

请求字段：

| 字段名称 | 字段类型 | 是否必填 | 标识       |
| -------- | -------- | -------- | ---------- |
| name     | String   | 是       | 采集点名称 |
| phone    | String   | 是       | 电话       |
| password | String   | 是       | 密码       |
| csId     | int      | 是       | 采集点ID   |


返回数据：

| 字段名称  | 字段类型 | 标识                     |
| --------- | -------- | ------------------------ |
| errorCode | int      | 返回200为成功,其他为失败 |
| message   | string   | 请求返回消息             |
````
{
    "errorCode": 200,
    "message": "录入成功",
}
````
